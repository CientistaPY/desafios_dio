Projeto feito em 4 Partes: 

Parte 1 - O Arquivo README.txt explicando sobre o Projeto.

Parte 2 - O Arquivo "SistemsBancarioComSQLite.py" contendo o código adaptado do Sistema Bancário com dados persistentes via SQLite, com comentários detalhados. 

Parte 3 - O Arquivo "banco_de_dados.db" contendo o banco de dados utilizado no "SistemsBancarioComSQLite.py"

Parte 4 - Arquivo SistemaBancarioComMangoDB.py contendo o código adaptado do Sistema Bancário com dados persistentes via Atlas clusterizado do MongoDB, com comentários detalhados. 





OBSERVAÇÕES:

Todo o código foi readaptado para adequar-se ao desafio proposto modificando o sistema bancário que salva as informações na memória local, para um sistema que utiliza Banco de dados relacional e não relacional, utilizando o SQLAlchemy em um código e o PYMongo em outo.
